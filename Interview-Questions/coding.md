Great idea! Here are coding questions appropriate for 3 years iOS experience:

### Easy Level (Warm-up)

1. **FizzBuzz with a twist**: Write a function that prints numbers 1-100, but for multiples of 3 print "Fizz", multiples of 5 print "Buzz", multiples of both print "FizzBuzz". Make it generic to accept any two numbers and strings.

2. **Two Sum**: Given an array of integers and a target, return indices of two numbers that add up to target.

3. **Palindrome Check**: Check if a string is a palindrome, ignoring spaces and case.

4. **Reverse String**: Reverse a string without using built-in reverse() method.

5. **Find Duplicates**: Find all duplicate elements in an array.

### Medium Level (Core Skills)

6. **Valid Parentheses**: Check if a string with brackets (), {}, [] is valid (properly closed and nested).

7. **Merge Sorted Arrays**: Merge two sorted arrays into one sorted array efficiently.

8. **Fibonacci Sequence**: Implement Fibonacci using recursion, iteration, and memoization. Compare performance.

9. **Binary Search**: Implement binary search on a sorted array. What's the time complexity?

10. **Linked List Cycle**: Detect if a linked list has a cycle in it.

11. **Anagram Check**: Check if two strings are anagrams of each other.

12. **Group Anagrams**: Given an array of strings, group anagrams together.

13. **Remove Duplicates from Sorted Array**: Remove duplicates in-place and return new length.

14. **Maximum Subarray Sum**: Find the contiguous subarray with the largest sum (Kadane's algorithm).

15. **Rotate Array**: Rotate an array to the right by k steps.

### iOS-Specific Coding Questions

16. **Custom Observable Property Wrapper**: Create a property wrapper that notifies observers when value changes.

17. **LRU Cache**: Implement a Least Recently Used cache with get and put operations in O(1).

18. **Debounce Function**: Implement a debounce mechanism for search text field (waits until user stops typing).

19. **Generic Network Layer**: Design a generic network client that handles GET/POST requests with Codable.

20. **Image Downloader with Cache**: Implement an async image downloader with memory and disk caching.

21. **Custom JSON Parser**: Parse nested JSON with custom key mapping without using Codable.

22. **Thread-Safe Singleton**: Implement a thread-safe singleton pattern with lazy initialization.

23. **Observer Pattern**: Implement a custom observer/notification system without using NotificationCenter.

24. **Async Image Loader**: Create an AsyncImage-like view that loads images from URL with placeholder and error handling.

25. **Deep Copy**: Implement deep copy for a complex object graph with circular references.

### Advanced Level (Problem Solving)

26. **Longest Substring Without Repeating Characters**: Find the length of longest substring without repeating characters.

27. **Validate Binary Search Tree**: Check if a binary tree is a valid BST.

28. **Level Order Traversal**: Traverse a binary tree level by level.

29. **Merge K Sorted Lists**: Merge k sorted linked lists into one sorted list.

30. **Word Ladder**: Transform one word into another, changing one letter at a time (BFS problem).

31. **LongestCommon Subsequence**: Find the longest common subsequence between two strings (DP).

32. **Coin Change**: Given coins and amount, find minimum coins needed (Dynamic Programming).

33. **Top K Frequent Elements**: Find k most frequent elements in an array.

34. **Serialize/Deserialize Binary Tree**: Convert binary tree to string and back.

35. **Design Hit Counter**: Design a hit counter that counts hits in the past 5 minutes.

### System Design Coding

36. **Rate Limiter**: Implement a rate limiter that allows n requests per time window.

37. **Pagination Manager**: Build a pagination system that prefetches next page before user reaches end.

38. **Offline Queue**: Implement a queue that stores failed API requests and retries when online.

39. **Auto-Complete/Trie**: Implement a Trie data structure for autocomplete suggestions.

40. **Notification Scheduler**: Design a local notification scheduler with priority queue.

### SwiftUI-Specific Coding

41. **Custom View Modifier**: Create a reusable shimmer loading effect modifier.

42. **Generic List with Pull-to-Refresh**: Build a generic list component with pull-to-refresh and pagination.

43. **Custom Binding**: Create a custom Binding that transforms values (e.g., String to Int).

44. **Cancellable Task Manager**: Implement a task manager that cancels previous tasks when new one starts.

45. **State Machine**: Implement a loading state machine (idle, loading, success, failure) with SwiftUI.
